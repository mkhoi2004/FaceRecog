1. Gioi thieu he thong

FaceIDApp la ung dung cham cong WinForms su dung nhan dien khuon mat.

Muc tieu

  • Cham cong tu dong bang webcam.
  • Quan ly nhan vien, ca lam, nghi phep.
  • Luu audit log va bao cao theo ngay/thang.

Cong nghe chinh

  • .NET Framework 4.6.1
  • DlibDotNet + OpenCvSharp
  • SQLite

Luu y quan trong

  • Chi ho tro x64.
  • Can 2 file model .dat trong thu muc models.
  • Khong chay duoc neu thieu native DLL.
