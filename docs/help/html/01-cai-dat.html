2. Cài đặt và khởi động lần đầu

⚠️ LƯU Ý QUAN TRỌNG: Ứng dụng chỉ chạy trên nền tảng Windows 64-bit (x64). Vui lòng không cài đặt trên Windows 32-bit.

Bước 1: Kiểm tra môi trường hệ thống

  • Hệ điều hành: Windows 10 hoặc Windows 11 (phiên bản 64-bit).
  • Thư viện bổ sung: Yêu cầu cài đặt Visual C++ Redistributable 2015-2022 x64 để các thư viện AI có thể hoạt động.
  • Phần cứng: Webcam (tích hợp hoặc USB) đang hoạt động bình thường và không bị ứng dụng khác chiếm dụng.

Bước 2: Cấu trúc thư mục ứng dụng

Khi giải nén/cài đặt, hãy đảm bảo thư mục ứng dụng có đầy đủ các thành phần sau:

  • FaceIDApp.exe: File thực thi chính.
  • models/: Thư mục chứa 2 file dữ liệu AI (.dat). Đây là "bộ não" của hệ thống nhận diện.
  • net461/: Chứa các thư viện native (DLL) cần thiết.
📍 LƯU Ý VỀ ĐƯỜNG DẪN (Unicode Bug):
Các thư viện nhận diện khuôn mặt (Dlib) không hỗ trợ đường dẫn có chứa ký tự tiếng Việt có dấu.

Ví dụ:
❌ D:\Dự án\FaceIDApp\ (KHÔNG CHẠY ĐƯỢC)
✅ D:\Projects\FaceIDApp\ (CHẠY TỐT)

Bước 3: Khởi động và Cấu hình

  1. Chạy file FaceIDApp.exe.
  2. Trong lần đầu tiên, hệ thống sẽ tự động khởi tạo Cơ sở dữ liệu SQLite (face_attendance.db).
  3. Nếu ứng dụng yêu cầu quyền Admin để tạo file, hãy chuột phải vào file EXE và chọn "Run as Administrator".

Bước 4: Kiểm tra nhanh (Smoke Test)

Sau khi mở ứng dụng, hãy thực hiện các bước sau để đảm bảo mọi thứ đã sẵn sàng:

  1. Đăng nhập bằng tài khoản mặc định (admin/admin123).
  2. Vào tab "Chấm công" để kiểm tra camera đã lên hình chưa.
  3. Thử đăng ký khuôn mặt cho 1 nhân viên mẫu.
