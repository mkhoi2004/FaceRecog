8. Đóng gói Release x64

Hướng dẫn này dành cho kỹ thuật viên hoặc quản trị viên khi cần triển khai ứng dụng sang máy tính khác hoặc tạo bản cài đặt chính thức.

Quy trình build bản Release

Nếu bạn có mã nguồn, hãy sử dụng lệnh sau trong Terminal (hoặc Developer PowerShell) để tối ưu hóa hiệu năng:

dotnet build .\FaceIDApp\FaceIDApp.csproj -c Release -p:Platform=x64 --nologo

Các thành phần bắt buộc khi triển khai

Sau khi build, bạn cần thu thập đủ các file sau để bộ cài hoạt động ổn định:

  • FaceIDApp.exe: File chạy chính.
  • FaceIDApp.exe.config: File cấu hình hệ thống (Chuỗi kết nối, Tolerance).
  • Thư mục models/: Phải chứa 2 file .dat (ResNet model và Shape Predictor).
  • Thư mục net461/: Chứa các file thư viện trung gian và native DLL (DlibDotNetNative.dll, OpenCvSharpExtern.dll,...).
  • Database/face_attendance_v3.sql: Dùng để hệ thống tự tạo cấu trúc CSDL nếu máy mới chưa có dữ liệu.
⚠️ LƯU Ý KHI DỌN DẸP DỮ LIỆU:
Trước khi gửi bộ cài cho khách hàng hoặc nhân viên, hãy đảm bảo KHÔNG bao gồm các file dữ liệu test sau:
  • face_attendance.db (Database đang dùng thử).
  • Thư mục face/ hoặc img/ (Chứa ảnh đăng ký của bạn).
  • Các file .log trong thư mục logs.

Yêu cầu máy trạm (Client)

  • Cài sẵn Microsoft .NET Framework 4.6.1 trở lên.
  • Cài đặt Visual C++ Redistributable 2015-2022 (x64).
  • Webcam có độ phân giải tối thiểu 720p để đảm bảo độ chính xác của AI.

Checklist triển khai thành công

  • App mở lên không báo lỗi "Missing DLL".
  • Camera khởi động thành công và hiển thị mượt mà (640x480).
  • Thử đăng ký 1 khuôn mặt và chấm công thành công.
  • Xuất báo cáo thử để kiểm tra quyền ghi file vào thư mục.
